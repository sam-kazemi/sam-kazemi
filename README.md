# Hey, I'm Sam 👋

UIUC's Beckman Institute. My background is in electrical engineering,
so technically I've just moved from debugging circuits to debugging
the wetware that designed them.

  (the neurons are doing their best)
- Previously: an **EEG brain-computer interface** that let users control
  a cycling rehab system with their thoughts; pedaling, but make it telepathic
